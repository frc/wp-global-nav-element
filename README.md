# WordPress Global Navigation Element

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.5 and use [Angular Elements](https://angular.io/guide/elements).

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Build elements

Run `npm run build:elements` to build the project element. The production ready build element will be stored in the `elements/` directory.
